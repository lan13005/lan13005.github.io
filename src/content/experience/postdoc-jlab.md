- Developing a declarative framework for smooth non-parametric statistical models using Python, JAX, and OpenMPI with Gaussian Processes and Variational Inference, enabling Bayesian inference over O(10^6) free parameters. ([GitHub](https://github.com/fmkroci/iftpwa))
- Built an end-to-end analysis pipeline generating physics events from a model, emulating detector efficiency via normalizing flows, and interfacing with both Bayesian- and frequentist-based optimization frameworks for rapid closure tests. ([GitHub](https://github.com/lan13005/PyAmpTools))
- Created Python bindings for the collaboration's C++ analysis software, providing a more dynamic interaction that simplifies workflows.
- Created and maintain user-friendly documentation for the above software to ensure long-term sustainability and guide future developers.
